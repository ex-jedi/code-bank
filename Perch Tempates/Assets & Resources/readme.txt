Herein are some things for seting up Perch sites.

Setup list
1. Setup Trello Board.

2. Buy Perch licence.

3. Setup Git repo.
   Setup on Bitbucket first and clone into local file.

4. Configure Localhost.

5. Setup local database.

6. Add site files including Perch CMS.

7. Perch Setup.

8. Initial Git commit & push.

9. Setup staging server database.

10. Configure Perch CMS.
    Delete Core gitignore file
    Create resources folder
    Copy database from local
    Delete Perch setup file.

11. Setup Deployment
    Setup dev & live sites
    Setup config files
    Setup live push to dev site
12. Push to Live Site.

13. Setup live database.

14. Configure Perch CMS.
